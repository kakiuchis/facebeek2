== README

## URL

https://facebeek.herokuapp.com/

## 機能

- Bootstrapデザイン
- topic機能（閲覧はフォローフォロアー間のみ、編集削除は投稿者のみ）
- topicに投稿者表示
- コメント機能
- コメント機能をajaxで実装
- フォロー、フォロワー機能
- ログイン機能（メール認証）
- プロフィール画像登録機能
- メッセージ機能（フォローフォローワー間のみ）
- Twitterログイン
- IDやパスワードなどの環境変数化
- aws運用(Unicorn nginx)→無料期間終了でHeroku移行
- Capistranoで自動デプロイ→無料期間終了で運用停止

## バージョン

ruby 2.3.0p0 (2015-12-25 revision 53290) [x86_64-linux]
Rails 4.2.3
